## Check Launch Arguments

```bash

```

## Start Mapping

```bash
cd /home/ryan/tb4_ws
source /opt/ros/jazzy/setup.bash
source install/setup.bash
ros2 launch sim map.launch.py
```

Optional overrides:

```bash
ros2 launch sim map.launch.py model:=lite rviz:=false
ros2 launch sim map.launch.py x:=0.0 y:=-5.8 yaw:=1.5708
```
