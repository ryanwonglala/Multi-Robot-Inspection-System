# Plan 20260531 Semantic Nav Check Package

## Goal

Create an isolated ROS2 package for validating semantic map coordinates with the real TurtleBot4 Gazebo/Nav2 stack.

## Package

```text
src/semantic_nav_check
```

## Intended Use

Start Gazebo, spawn TurtleBot4, load the manually saved map, start localization/Nav2/RViz, visualize semantic markers, and send selected semantic target poses from a console UI.

## Boundary

This package is only for coordinate validation. It does not modify `/opt/ros/jazzy`, and it stays separate from both `src/sim` and the dry-run experiment.
