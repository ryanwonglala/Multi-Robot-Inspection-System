# Cmd 20260531 Semantic Nav Check Package

## Build

```bash
cd /home/ryan/tb4_ws
colcon build --packages-select semantic_nav_check
source install/setup.bash
```

## Launch Verification Stack

```bash
ros2 launch semantic_nav_check verify.launch.py
```

## Run Target Console

Open a second terminal:

```bash
cd /home/ryan/tb4_ws
source install/setup.bash
ros2 run semantic_nav_check target_console.py --world-model install/semantic_nav_check/share/semantic_nav_check/config/world_model.yaml
```

## Launch Argument Check

```bash
cd /home/ryan/tb4_ws
source install/setup.bash
ros2 launch semantic_nav_check verify.launch.py --show-args
```
