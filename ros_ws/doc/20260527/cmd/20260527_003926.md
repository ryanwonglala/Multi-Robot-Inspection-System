## Rebuild

```bash
cd /home/ryan/tb4_ws
source /opt/ros/jazzy/setup.bash
colcon build
source install/setup.bash
```

## Launch With New Default Yaw

```bash
ros2 launch sim map.launch.py
```

## Override Yaw If Needed

```bash
ros2 launch sim map.launch.py yaw:=1.5708
ros2 launch sim map.launch.py yaw:=-1.5708
```

## Reliable Teleop Path

```bash
ros2 run teleop_twist_keyboard teleop_twist_keyboard
```

## Direct Velocity Test

Forward briefly:

```bash
ros2 topic pub --once /cmd_vel geometry_msgs/msg/Twist "{linear: {x: 0.15}, angular: {z: 0.0}}"
```

Stop:

```bash
ros2 topic pub --once /cmd_vel geometry_msgs/msg/Twist "{linear: {x: 0.0}, angular: {z: 0.0}}"
```
