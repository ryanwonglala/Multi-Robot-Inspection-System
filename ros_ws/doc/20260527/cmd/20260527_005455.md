## Confirm Topic Type

With simulation running:

```bash
ros2 topic info /cmd_vel
```

Expected type:

```text
geometry_msgs/msg/TwistStamped
```

## Echo GUI Command

```bash
ros2 topic echo /cmd_vel --once
```
